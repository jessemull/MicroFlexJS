Class: Random

Utilities.Random

new Random()

This class provides static utility helper methods for generating random wells, well groups. well sets, plates and stacks. The random class can return random instances of the following objects and primitives:
»  Integers
»  Float Arrays
»  Integer Arrays
»  Row Strings
»  Well Indices
»  Well Index Arrays
»  Wells w/ Float Data
»  Wells w/ Integer Data
»  Well Arrays
»  Well Sets
»  Well Set Arrays
»  Well Groups
»  Well Group Arrays
»  Plates
»  Plate Arrays
