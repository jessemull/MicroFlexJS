Class: MicroXML

IO.MicroXML

new MicroXML()

This class formats stack, plate, well set and well objects into a condensed and easy to read XML format. Wells, well groups and plates are housed within typed hash set objects. This class converts the hash sets to simple objects which are easier to read and parse.
