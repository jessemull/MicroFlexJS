Class: MicroJSON

IO.MicroJSON

new MicroJSON()

This class formats stack, plate, well set and well objects into a condensed and easy to read JSON format. The MicroJSON.stringify and MicroJSON.parse functions mirror JSON.stringify and JSON.parse, accepting replacer/space and reviver arguments. Wells, well groups and plates are housed within typed hash set objects. This class converts the hash sets to simple objects which are easier to read and parse.
